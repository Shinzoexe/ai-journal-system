# AI Journal System

An AI-powered journaling application that analyzes emotions and extracts keywords from journal entries.

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Features
- Create journal entries
- Emotion detection
- Keyword extraction
- AI insights

## API Endpoints

POST /api/journal  
GET /api/journal/:userId  
GET /api/journal/insights/:userId

## Run Locally

Backend
cd backend
npm install
node server.js

Frontend
cd frontend
npm install
npm start
